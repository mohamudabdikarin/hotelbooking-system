# Hotel Booking System

This is simple system designed to manage the operations of the hotel such as managing Rooms, bookings, customers, payments, and Users.

## Technology Used:

Frontend: Reactjs + tailwindcss
-Backend: Nodejs with Express
-Database: MongoDB

# here is how to install and run the project

1. Clone the repository: `git clone https://github.com/mohamudabdikarin/hotelbooking-system.git`
2. `cd hotelbooking-system` to change the directory of your terminal
3. `cd backend` and run `npm install` then `npm run dev`
4. `cd frontend` and run `npm install`
5. create content from .env.example and paste in new file called `.env`
6. Run the frontend with `npm run dev`
